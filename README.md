# 🏁 Formula One Winner Predictor

A Python project that analyzes recent Formula 1 seasons and uses machine learning to predict future race winners. The project focuses on the current technical regulations era, to ensure data relevance and accuracy.

## 📌 Purpose

This project is a personal learning journey in:
- Data engineering and analysis with **DuckDB** and **Pandas**
- Machine learning using **scikit-learn**
- Visualizing insights with **Plotly**, **Matplotlib**, or **Seaborn**

It also serves as a foundation for building more complex sports prediction models in the future.

## 🧠 Key Features

- Clean and filter race data using SQL via DuckDB
- Explore trends and patterns using interactive plots
- Train ML models (e.g. Logistic Regression, Random Forest) on recent seasons
- Evaluate model accuracy and predict upcoming race outcomes

## 🛠️ Technologies Used

- 🐍 Python
- 🦆 DuckDB
- 📊 Pandas, Plotly, Seaborn
- 🤖 scikit-learn
- 📁 VS Code
